# Weather Study – Lienz (10-Minute Climate Data)

Dieses Projekt lädt und analysiert historische 10-Minuten-Klimadaten (CSV) von GeoSphere Austria und visualisiert sie mit Pandas und Matplotlib.

---

## 📥 Datenquelle

Die historischen Stationsdaten können hier heruntergeladen werden:

https://dataset.api.hub.geosphere.at/app/frontend/station/historical/klima-v2-10min

Dort:

1. Station auswählen (z.B. Lienz)
2. Zeitraum definieren
3. CSV herunterladen
4. Lokal im Projekt analysieren
